Description

A real-time hand gesture recognition system that uses MediaPipe to detect hand landmarks and a Random Forest model to classify gestures from webcam input.

## Features
Real-time webcam detection
Hand landmark extraction
Gesture classification (0–9)
Easy to run

##Technologies Used
Python
OpenCV
MediaPipe
Scikit-learn

# How to Run
pip install mediapipe opencv-python scikit-learn numpy
python hand_gesture_vs.py

## Output
Displays hand landmarks
Predicts gesture in real-time
Shows output as numbers
